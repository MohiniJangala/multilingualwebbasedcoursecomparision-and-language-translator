# Multilingual Learning Platform

## Overview

The Multilingual Learning Platform is a web-based application designed to enhance online learning accessibility and decision-making. The platform integrates two major modules:

1. Multilingual Video Translation System
2. Course Comparison & Analytics Platform

This system helps users translate educational video content into multiple languages and compare online courses across various learning platforms using analytics and recommendation techniques.

---

## Features

### Multilingual Video Translation System

- Video and audio processing
- Speech-to-Text conversion
- Multilingual translation
- Subtitle generation
- Improved accessibility for regional language learners

### Course Comparison & Analytics Platform

- Compare courses across multiple learning platforms
- Analyze ratings, reviews, enrollments, and course quality
- Course recommendation support
- Interactive analytics dashboard
- Data-driven decision making for learners
